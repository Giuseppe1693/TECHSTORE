TechStore

TechStore è un’applicazione e-commerce sviluppata interamente in React, progettata per dimostrare flussi completi di registrazione e autenticazione utente. L’accesso all’app è consentito solo agli utenti registrati, garantendo un’esperienza d’uso coerente e sicura. Una volta autenticati, gli utenti possono aggiungere o rimuovere i prodotti dal carrello e interagire con un’interfaccia responsive e user-friendly.

Competenze dimostrate

* Implementazione di flussi di registrazione e login lato frontend
* Gestione dello stato dell’applicazione (carrello e prodotti)
* Consumo e visualizzazione dinamica dei dati
* Progettazione di interfaccia responsive con Tailwind CSS
* Strutturazione di un’applicazione React modulare e manutenibile

Tecnologie utilizzate

* React per lo sviluppo del frontend
* JavaScript per la logica applicativa
* Tailwind CSS per lo styling e layout responsive
